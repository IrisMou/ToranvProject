# 微球App - 专业足球预约平台

  "微球App - 专业足球预约平台"是一款关于足球的商城+场馆的应用。

    商城：关于足球、足球鞋、足球配件等相关足球运动的商品；
         应用内直接可下单，到店取货。

    场馆：关于足球的优选场馆；
         应用内下单预约，预约时间到达场馆进行消费。

    每个场馆均有教练，用户可直接预约场馆或预约场馆下的教练。

    喜欢足球运动的宝宝们，快行动起来。。。。
